Problema: Devo creare un generatore di password partendo da informazioni inserite dall'utente (nome,cognome e colore preferito). In fine devo rendere visibile il risultato.

1. Dare all'utente la possibilità di inserire nome
    1.1 Creare variabile nome
    1.2 Collegare la variabile al valore prompt
2. Dare all'utente la possibilità di inserire cognome
    2.1 Creare variabile cognomeome
    2.2 Collegare la variabile al valore prompt
3. Dare all'utente la possibilità di inserire colore preferito.
    3.1 Creare variabile colore preferito
    3.2 Collegare la variabile al valore prompt
4. Creare la variabile password concatenando i risultati tra loro e con il numero 21
5. Mostrare il risultato della variabile password richiamando l'id dell'elemento nell'html.
